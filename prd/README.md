## 目录结构

| 目录 | 用途 |
|------|------|
| `00-基础数据/` | 项目背景、进度记录、成员名册、需求池 |
| `00-调研/` | 技术调研、决策记录（ADR） |
| `01-首页/` | 首页需求 |
| `02-主界面/` | 导航、菜单需求 |
| `03-男德通/` | AI 助手需求 |
| `04-德塔/` | 德塔虚拟世界（需求/设计/技术方案/部署/changelog/bug-log） |
| `05-美术设计/` | 美术资源规范 |
| `06-师德墙/` | 师德墙模块（PRD/changelog/bug-log） |

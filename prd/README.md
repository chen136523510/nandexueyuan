# 男德学院 - PRD 文档库

> 产品需求与技术文档总入口。

## 目录结构

| 目录 | 用途 |
|------|------|
| `01-需求调研/` | 需求模块（首页/导航/菜单等）、项目背景、进度记录 |
| `02-技术架构/` | 技术选型、系统架构、部署方案 |
| `03-功能模块/` | 各功能模块 PRD（含业务契约、状态机、异常分支） |
| `04-接口契约/` | API 契约规范（路径、入参、出参、错误码） |

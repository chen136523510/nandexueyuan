# 投票 PRD

## 1. Why-Who-What
- **业务背景**：群体决策，少数服从多数
- **目标用户**：约 20 名注册成员
- **功能范围**：发起投票、投票、查看结果

## 2. 功能清单
| 功能 | 优先级 | 状态 |
|------|--------|------|
| 发起投票 | P0 | 待开发 |
| 投票 | P0 | 待开发 |
| 查看结果 | P0 | 待开发 |
| 截止时间 | P1 | 待开发 |

## 3. 异常分支（MECE）
- [ ] 投票：重复投票
- [ ] 投票：已截止
- [ ] 投票：选项不存在
- [ ] 结果：无人投票空状态
- [ ] 发起：选项不足 2 个

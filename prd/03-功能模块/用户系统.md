## 4. 异常分支（MECE）
- [ ] 注册：用户名已存在
- [ ] 注册：邮箱格式错误
- [ ] 登录：账号不存在
- [ ] 登录：密码错误
- [ ] 鉴权：token 过期/无效
- [ ] 鉴权：无权限访问他人资料
- [ ] 网络：弱网/断网降级
- [ ] 空状态：首次进入无个人信息

# 论坛 PRD

## 1. Why-Who-What
- **业务背景**：朋友间发帖交流
- **目标用户**：约 20 名注册成员
- **功能范围**：发帖、评论、讨论

## 2. 功能清单
| 功能 | 优先级 | 状态 |
|------|--------|------|
| 发帖（文字） | P0 | 待开发 |
| 帖子列表 | P0 | 待开发 |
| 帖子详情 | P0 | 待开发 |
| 评论 | P1 | 待开发 |
| 删除自己的帖/评论 | P1 | 待开发 |

## 3. 异常分支（MECE）
- [ ] 发帖：内容为空
- [ ] 发帖：超长内容限制
- [ ] 列表：空状态（无帖子）
- [ ] 列表：分页边界
- [ ] 详情：帖子不存在/已删除
- [ ] 评论：评论对象不存在
- [ ] 权限：删除他人内容

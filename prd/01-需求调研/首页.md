# 首页需求

## 定位
网站入口，展示学院形象与未来规划预告。

## 目标用户
所有访客（含未登录）

## 功能范围
| 功能 | 优先级 | 状态 |
|------|--------|------|
| Hero 区（院名 + 院训 + 简介） | P0 | 已完成 |
| 即将上线板块（群聊数据看板 + 群友高光时刻） | P0 | 已完成 |
| 页脚（版权声明） | P0 | 已完成 |
| 彩蛋：右上角隐藏照片，点击弹出 | P1 | 已完成 |
| 群聊数据看板（实际功能） | P2 | 待开发 |
| 群友高光时刻（实际功能） | P2 | 待开发 |

## 首页结构
1. **Hero** — 院名「男德学院」+ 院训「修身 齐家 摸鱼 搞笑」+ 简介
2. **即将上线** — 两个预告卡片：
   - 群聊数据看板：聊天统计、活跃度趋势、话痨排行榜
   - 群友高光时刻：名场面存档、金句合集、社死瞬间
3. **页脚** — 版权声明

## 彩蛋需求
- 丘序明照片固定在页面右上角，大部分隐藏，仅露出一角
- 鼠标悬停时透明度变化提示可点击
- 点击后弹出完整照片 + 文字「恭喜找到彩蛋」
- 点击遮罩或关闭按钮关闭弹窗

## 异常分支
- [ ] 图片加载失败 → 隐藏彩蛋入口
- [ ] 移动端彩蛋尺寸缩小，避免遮挡内容

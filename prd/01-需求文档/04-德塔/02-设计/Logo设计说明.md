## 2. 德塔（Delta）的含义与引申

### 2.1 字面含义

| 层面 | 含义 |
|------|------|
| 谐音 | 德塔 = Delta（Δέλτα），希腊字母 Δ |
| 中文拆解 | 德 = 男德学院的"德"；塔 = 像素世界的高塔/据点 |
| 英文 NDO | Nande Online = 男德在线 |

### 2.2 数学/科学引申

| 符号 | 含义 | 与德塔的关联 |
|------|------|-------------|
| Δ（大写） | 变化量、增量 | 进入德塔 = 进入改变的世界，"Δ 世界" |
| δ（小写） | 微小变化 | 每个成员都是像素世界的微小变量 |
| 三角形 | 稳定、坚固 | 德塔是男德学院的稳定据点 |

### 2.3 文化引申

| 含义 | 说明 |
|------|------|
| 三角洲 | 河流入海口的三角洲，暗示"汇入新世界" |
| 第四维度 | Δ 是第四个希腊字母，暗示"三维之外的第四维度" |
| 入口符号 | 三角形天然有"箭头/入口/传送门"的视觉意象 |

## 3. Logo 设计方案

### 3.1 方案 A：Δ 内嵌 NDO（推荐）

```
        ▲
       N D
        O
```

- 大三角 Δ 作为外轮廓
- 内部嵌入 NDO 三个字母
- 像素风边缘处理

### 3.2 方案 B：NDO 变体

```
    N   D   O
    ██ ██  ██
    █ ██ █ ██
    ██ ██ ██
```

- 三个字母以像素块风格呈现
- O 字母设计为菱形/三角形暗示
- 适合作为纯文字 Logo

### 3.3 方案 C：纯 Δ 符号

```
        ▲
       ▲ ▲
      ▲ ▲ ▲
```

- 三个小三角形组成大三角形（像素风）
- 下方 NDO 小字
- 极简，高辨识度
- 适合作为 favicon、应用图标

### 3.4 配色方案

| 用途 | 配色 | 色值建议 |
|------|------|----------|
| 主色 | 像素绿 | #4CAF50 或 #00FF66 |
| 辅色 | 深蓝/黑 | #1A1A2E |
| 强调色 | 金色 | #FFD700 |
| 背景 | 像素暗色 | #2D2D2D |

### 3.5 像素风处理

- Logo 在 32x32 或 64x64 像素网格中设计
- 边缘锯齿化（像素风特征）
- 保留 1px 间隔的像素空隙

## 4. 使用场景

| 场景 | 规格 | 建议 |
|------|------|------|
| 导航栏入口图标 | 32x32 px | 方案 C（纯 Δ） |
| 德塔加载页 | 256x256 px | 方案 A（Δ + NDO） |
| 页面标题 | 任意 | 方案 B（NDO 文字） |
| favicon | 16x16 / 32x32 px | 方案 C（纯 Δ） |
| 游戏内传送门 | 32x32 瓦片 | 方案 C（Δ 像素块） |

## 6. 设计原则

1. **像素优先**：Logo 必须是像素风，与德塔整体美术风格一致
2. **可缩放**：从小至 16x16（favicon）到大至 256x256（加载页）都要清晰
3. **Δ 符号核心**：无论哪个方案，Δ 三角形是核心识别元素
4. **简洁有力**：不需要多余的装饰，像素风格本身就有辨识度

# 德塔（NDO）目录结构规范

> 版本：V1 | 日期：2026-07-13 | 状态：已确认

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| 游戏代码与网页代码隔离 | Phaser 场景/对象/系统的组织逻辑与 Vue 组件完全不同，混在一起互相干扰 |
| 游戏后端独立进程 | Colyseus 是独立进程（:2567），生命周期、依赖、部署节奏都和 Express(:3000) 不同 |
| GameView.vue 是唯一接触点 | Vue 只负责挂载 Canvas + 弹窗桥接，游戏逻辑全在 `game/` 内 |
| 资源放 public/ | Vite 对 `public/` 原生服务，Phaser 可直接 URL 加载 |
| 共享层提取到根目录 | game 和 game-server 都要引用，放任一方都不合适 |

## 3. 各层职责边界

### 3.1 `game/` vs `src/`

| 职责 | 归属 |
|------|------|
| Phaser 场景、游戏对象、物理碰撞 | `game/` |
| 键盘输入、网络同步逻辑 | `game/` |
| Vue 路由、页面布局 | `src/` |
| NPC 弹窗 UI、外观选择面板 | `src/`（Vue 组件） |
| Phaser 挂载 + 事件桥接 | `src/views/GameView.vue`（唯一接触点） |

### 3.2 `game-server/` vs `server/`

| 职责 | 归属 |
|------|------|
| 游戏房间管理、状态同步 | `game-server/` |
| 玩家位置验证、移动指令 | `game-server/` |
| JWT 验证（复用密钥） | `game-server/src/lib/auth.js` |
| AI 助手、公告、用户 CRUD | `server/`（不动） |
| SQLite 数据库 | `server/`（不动） |

### 3.3 `shared/` 的使用

`shared/` 文件同时被 `game/` 和 `game-server/` 引用：

| 文件 | 谁引用 | 引用内容 |
|------|--------|----------|
| `constants.js` | game + game-server | 地图尺寸、物理参数、交互距离、移动指令枚举 |
| `avatars.js` | game（外观选择面板） + game-server（验证 avatarId） | 外观列表 |
| `npcs.js` | game（渲染 NPC/物品） + game-server（可选：服务器验证交互） | NPC/物品配置 |

## 4. 命名规范

### 4.1 文件命名

| 类型 | 命名 | 示例 |
|------|------|------|
| Phaser 场景 | PascalCase + Scene 后缀 | `WorldScene.js` |
| 游戏对象 | PascalCase | `Player.js` |
| 系统 | PascalCase + System 后缀 | `NetworkSystem.js` |
| Schema | PascalCase + State 后缀 | `PlayerState.js` |
| 常量/配置 | camelCase | `constants.js` |
| 共享配置 | camelCase | `avatars.js` |

### 4.2 资源命名

```
{类型}_{名称}_{方向}_{帧号}.png

示例：
tile_grass_00.png                    # 草地瓦片
sprite_player_default_r_01.png       # 玩家默认外观 右向 帧1
sprite_npc_nandetong_idle.png        # NPC男德通 站立
portrait_nandetong.png               # 男德通立绘
item_board_notice.png                # 公告牌物品
```

## 5. .gitignore 策略

| 路径 | 策略 | 原因 |
|------|------|------|
| `game/` | 提交 | 游戏源码 |
| `game-server/src/` | 提交 | 服务器源码 |
| `game-server/.env` | 忽略 | 各机器独立（已在全局 `.env` 规则中） |
| `shared/` | 提交 | 共享配置 |
| `public/game/` | 提交 | 美术资源（体积小，~100MB） |
| `public/game/audio/` | 忽略内容，保留 .gitkeep | 音频文件体积大，各机器独立 |
| `game-server/node_modules/` | 忽略 | 依赖目录（已在全局规则中） |

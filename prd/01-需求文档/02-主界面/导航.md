# 导航需求

## 定位
全局导航，在各页面间切换。

## 目标用户
所有登录用户

## 功能范围
| 功能 | 优先级 | 状态 |
|------|--------|------|
| 顶部导航栏 | P0 | ✅ 已完成 |
| 导航项：首页/男德通/师德墙/男通讯录/德塔 | P0 | ✅ 已完成 |
| 用户头像/昵称下拉菜单 | P1 | 待开发 |
| 移动端导航折叠/展开 | P1 | 待开发 |

## 导航项说明

| 顺序 | 名称 | 路由 | 说明 |
|:----:|------|------|------|
| 1 | 首页 | `/home` | 首页 |
| 2 | 男德通 | `/chat` | AI 问答（群聊数据检索） |
| 3 | 师德墙 | `/wall` | 校园墙/朋友圈（图文动态/评论/点赞） |
| 4 | 男通讯录 | `/admin` | 成员管理（仅 admin/super_admin 可见） |
| 5 | 德塔 | `/nde` | 2D 虚拟世界 |

## 异常分支
- [x] 未登录访问需登录页面 -> 跳转登录页
- [x] token 过期 -> 自动跳转登录页
- [ ] 移动端导航折叠/展开状态

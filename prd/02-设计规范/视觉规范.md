# 男德学院 · 视觉设计规范

> 「熟人线上小窝」——松弛、归属、平等。本文档定义网站统一的视觉语言,所有页面遵循此规范。

---

## 一、设计理念

| 关键词 | 含义 |
|--------|------|
| 松弛 | 低对比度、柔和不疲劳,无压迫感 |
| 归属 | 熟人社区温度,非官方距离感 |
| 平等 | 弱化等级差异,突出平等交流 |
| 干净 | 大面积留白,无多余装饰 |

**整体传递**:温润松弛的「熟人线上小窝」,操作路径短,发帖回复门槛低,符合随手刷随手聊的习惯。

---

## 二、配色系统(莫兰迪低饱和)

### 2.1 核心色板

| 令牌 | 色值 | 用途 |
|------|------|------|
| `--md-primary` | `#A8C5A0` | 主色(鼠尾草绿)——按钮、链接、强调 |
| `--md-primary-hover` | `#94B48C` | 主色 hover |
| `--md-primary-bg` | `#EEF3EC` | 主色浅底(tag、选中态) |
| `--md-secondary` | `#AEC2CF` | 辅色(奶蓝)——次要按钮、信息提示 |
| `--md-accent` | `#D4A574` | 点缀(暖赭)——徽章、高亮(克制使用) |
| `--md-bg` | `#F5F2EC` | 页面背景(温暖米白) |
| `--md-bg-card` | `#FFFFFF` | 卡片背景 |
| `--md-bg-soft` | `#FAF8F3` | 次级背景(嵌套区域) |
| `--md-text` | `#4A4A4A` | 主文字(非纯黑,柔和深灰) |
| `--md-text-secondary` | `#8A8A8A` | 次文字(说明、时间) |
| `--md-text-disabled` | `#BFBFBF` | 禁用文字 |
| `--md-border` | `#E8E4DC` | 边框(细线条) |
| `--md-divider` | `#EDE9E1` | 分割线(更轻) |
| `--md-success` | `#A8C5A0` | 成功(复用主色) |
| `--md-warning` | `#D4A574` | 警告(复用点缀) |
| `--md-danger` | `#C9A0A0` | 危险(莫兰迪红,不刺眼) |

### 2.2 配色原则

- **低对比度**:文字与背景对比适中,不刺眼,适合长阅读
- **灰调主导**:所有颜色带灰调(莫兰迪),避免高饱和
- **克制点缀**:暖赭 `#D4A574` 仅用于徽章/高亮,不大面积使用
- **无纯黑纯白**:文字用 `#4A4A4A` 不用 `#000`,背景用 `#F5F2EC` 不用 `#FFF`(卡片除外)

## 三、字体系统

### 3.1 字体栈

```css
--md-font: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", "Segoe UI", sans-serif;
```

无衬线,系统字体优先(零加载,各平台原生体验)。

### 3.2 字号阶梯

| 令牌 | 值 | 用途 |
|------|----|------|
| `--md-fs-xs` | 12px | 辅助说明、时间戳 |
| `--md-fs-sm` | 13px | 次要文字、标签 |
| `--md-fs-base` | 14px | 正文(默认) |
| `--md-fs-md` | 16px | 强调正文 |
| `--md-fs-lg` | 18px | 卡片标题 |
| `--md-fs-xl` | 22px | 页面标题 |
| `--md-fs-2xl` | 28px | Hero 标题 |

### 3.3 行高

- 正文:`1.75`(宽松,舒适长阅读)
- 标题:`1.4`
- 紧凑场景(按钮、标签):`1.2`

### 3.4 字重

- 常规:`400`(正文)
- 中等:`500`(强调)
- 粗体:`700`(标题,少用)

## 四、间距系统

4 的倍数阶梯:

| 令牌 | 值 | 用途 |
|------|----|------|
| `--md-sp-1` | 4px | 微间距(图标与文字) |
| `--md-sp-2` | 8px | 紧凑间距 |
| `--md-sp-3` | 12px | 常用间距 |
| `--md-sp-4` | 16px | 卡片内边距 |
| `--md-sp-5` | 24px | 区块间距 |
| `--md-sp-6` | 32px | 大区块间距 |
| `--md-sp-7` | 48px | 页面级留白 |

## 五、圆角与阴影

### 5.1 圆角(中等,温润)

| 令牌 | 值 | 用途 |
|------|----|------|
| `--md-radius-sm` | 6px | 小元素(标签、按钮) |
| `--md-radius` | 8px | 输入框、按钮 |
| `--md-radius-lg` | 12px | 卡片 |
| `--md-radius-full` | 9999px | 头像、圆点 |

### 5.2 阴影(极轻,几乎无感)

| 令牌 | 值 | 用途 |
|------|----|------|
| `--md-shadow-sm` | `0 1px 3px rgba(74,74,74,0.04)` | 卡片默认 |
| `--md-shadow` | `0 2px 8px rgba(74,74,74,0.06)` | 卡片 hover |
| `--md-shadow-lg` | `0 4px 16px rgba(74,74,74,0.08)` | 弹窗、浮层 |

阴影色用 `rgba(74,74,74,...)` 而非 `rgba(0,0,0,...)`,更柔和。

## 六、组件规范

### 6.1 卡片

```
背景:var(--md-bg-card)
圆角:var(--md-radius-lg) (12px)
边框:1px solid var(--md-border)
阴影:var(--md-shadow-sm)
内边距:var(--md-sp-4) (16px) 或 20px 24px
hover:阴影升级为 var(--md-shadow)
```

### 6.2 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 背景 `--md-primary`,文字 `#FFF`,圆角 8px,hover `--md-primary-hover` |
| 次按钮 | 背景 `--md-bg-card`,边框 1px `--md-border`,文字 `--md-text` |
| 文字按钮 | 无背景无边框,文字 `--md-primary`,hover 浅底 `--md-primary-bg` |
| 危险按钮 | 背景 `--md-danger`,文字 `#FFF` |

- 高度:32px(sm)/ 36px(默认)/ 40px(lg)
- 内边距:横向 16px
- 过渡:`all 0.2s ease`

### 6.3 输入框

```
背景:var(--md-bg-card)
边框:1px solid var(--md-border)
圆角:var(--md-radius) (8px)
高度:36px(默认)
内边距:8px 12px
聚焦:border-color → --md-primary, box-shadow → 0 0 0 3px rgba(168,197,160,0.15)
字号:14px
```

### 6.4 标签/徽章

```
背景:var(--md-primary-bg) (浅绿)
文字:var(--md-primary-hover)
圆角:var(--md-radius-sm) (6px)
内边距:2px 8px
字号:12px
```

### 6.5 图标

- 库:Lucide(线性简约)
- 描边:1.5px
- 尺寸:16px(行内)/ 20px(按钮)/ 24px(卡片标题)
- 颜色:跟随文字 `currentColor`,不单独着色(除特殊场景)

### 6.6 头像

- 圆形(`--md-radius-full`)
- 尺寸:32px(导航)/ 48px(列表)/ 64px(个人资料)
- 默认:首字母 + 莫兰迪色底(稳定哈希)

## 七、布局规范

### 7.1 页面容器

- 最大宽度:`800px`(阅读舒适)或 `960px`(多列)
- 居中,两侧留白
- 移动端:全宽,padding 16px

### 7.2 间距

- 区块间距:`24px`(`--md-sp-5`)
- 卡片内边距:`16px` 或 `20px 24px`
- 大面积留白,不拥挤

### 7.3 导航栏

- 高度:52px
- 背景:`--md-bg-card` + 底部 1px 边框
- sticky 置顶

---

## 八、交互规范

### 8.1 过渡

- 默认:`all 0.2s ease`
- hover 反馈:颜色微变 + 阴影升级,无位移
- 禁用复杂动画(无弹跳、无 3D)

### 8.2 操作路径

- 轻量化:核心操作 ≤ 2 步
- 发帖/回复:低门槛,无复杂编辑器
- 符合"随手刷随手聊"

### 8.3 反馈

- 加载:轻量骨架屏或简单 spinner
- 成功:文字提示(绿色),无弹窗
- 失败:文字提示(危险色),可重试

---

## 九、氛围原则(平等交流)

- **弱化等级**:不显示用户等级徽章,角色仅功能权限用(如管理员编辑按钮)
- **突出平等**:统一头像样式,无 VIP 特殊效果
- **熟人温度**:文案口语化,不官方(如"发帖"叫"说点什么")
- **无距离感**:卡片圆角温和,不锐利;配色柔和不冷硬
